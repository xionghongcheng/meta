# 05 场景规则

同一个生活场景，在这个家庭里要有特殊解释。

## 吃饭

不是吃什么，而是：

- 妈妈能不能吃
- 爸爸会不会添乱
- 牙齿和身体状态能不能适应
- 这个家能不能安稳坐在一起

## 出门

不是去哪玩，而是：

- 路线是否安全
- 妈妈体力和状态行不行
- 爸爸会不会乱跑
- 是否需要提前设计规则
- 一次普通出门能不能完成

## 买东西

不是消费，而是：

- 妈妈有没有选择权
- 爸爸会不会分不清
- 女朋友如何把关审美和实用
- 我如何在安全和自由之间做平衡

## 看病

不是检查结果，而是：

- 如何让他们配合
- 如何压住情绪
- 如何建立长期健康秩序
- 如何把焦虑转成可执行安排

## 家务

不是分工，而是：

- 妈妈能不能参与
- 爸爸会不会越帮越乱
- 女朋友如何建立秩序
- 我如何用规则代替火气

## 过节

不是仪式感，而是：

- 这个家是否终于像普通家庭一样热闹一次
- 爸妈有没有参与
- 女朋友是否让家有了新生活气
- 我是否从旁观家庭变成建设家庭

## 老地方和怀旧

不是怀旧打卡，而是：

- 父母是否还能被看见为“曾经年轻过的人”
- 爸爸是否说出旧伤
- 妈妈是否通过表情和反应连接过去
- 我是否能从他们的过去理解他们的现在
