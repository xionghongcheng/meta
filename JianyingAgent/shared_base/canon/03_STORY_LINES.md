# 03 五条内容主线

V1 只保留五条长期主线，避免系统过散。

## 1. 妈妈重新生活

她重新吃饭、穿衣、表达、出门、变漂亮、参与家庭。

核心问题：

> 妈妈今天有没有重新获得一点选择和生活感？

## 2. 爸爸重新适应

他学习规则、少闯祸、照顾妈妈、接受城市和家庭秩序。

核心问题：

> 爸爸今天有没有少闯一点祸，或者露出一点笨拙温柔？

## 3. 小家重新建立

我和女朋友一起做饭、装修、收纳、过节、让家变得有烟火气。

核心问题：

> 这个家今天有没有更像一个能生活的家？

## 4. 我重新理解自己

我从逃离家庭，到接管家庭，再到修复自己。

核心问题：

> 我今天有没有更理解父母，也更理解自己一点？

## 5. 照护工具箱

用真实问题引出家庭好物、老人好物、健康安全和照护减负。

核心问题：

> 这个工具是否解决了我家真实照护问题？
