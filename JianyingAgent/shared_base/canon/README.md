# Shared Canon

这个目录是 JianyingAgent V2 的内容共享底座。

二阶段开始后，原先位于项目根目录 `content_brain/` 的长期内容资产已经正式迁入这里，`shared_base/canon/` 成为唯一主目录。

## 模块顺序

工具在做选题、脚本、拍摄清单、粗剪和复盘时，建议按下面顺序读取：

1. `01_ACCOUNT_CANON.md`
   账号世界观和核心命题。

2. `02_CHARACTER_PROFILES.md`
   四个核心人物的性格、能力、限制和叙事功能。

3. `03_STORY_LINES.md`
   五条长期内容主线。

4. `04_RELATIONSHIP_DETAILS.md`
   真实生活颗粒和关系规律。

5. `05_SCENE_RULES.md`
   吃饭、出门、买东西、看病、家务、过节等场景规则。

6. `06_COMMERCIAL_RULES.md`
   商业化边界、适合品类和植入原则。

7. `07_AGENT_ARCHITECTURE.md`
   内容大脑与各 Agent 的协作方式。

8. `08_SCRIPT_REVIEW_RUBRIC.md`
   `Script Writer Agent` 和 `Script Evaluator Agent` 共享的评分标准。

## 核心原则

> 大纲决定方向，细节决定真实，Agent 负责执行。

脚本链路采用固定顺序：先写后评，共享同一把尺子。
