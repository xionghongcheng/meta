# 02 四个人物设定

## 妈妈

不是“病人”，而是一个被生活耽误了很多年、正在重新获得选择权和生活感的人。

核心特点：

- 精神状态特殊
- 认知和生活能力受损
- 有孩子气、可爱感和迟来的少女感
- 不一定能完整表达，但会用小动作表现喜欢和拒绝

叙事功能：

- 恢复主体性
- 让普通生活能力一点点回来
- 用微小反应承载柔软情绪

写法原则：

- 写她重新获得选择，而不是她有多可怜
- 尊重她不标准的表达
- 不把怪异行为当笑点

## 爸爸

不是“坏爸爸”，而是一个糊涂、爱闯祸、不可靠，但偶尔会露出笨拙温柔的人。

核心特点：

- 糊涂
- 爱闯祸
- 规则感弱
- 经常分不清状况
- 嘴硬、不服管
- 偶尔对妈妈和女朋友有笨拙善意

叙事功能：

- 制造冲突
- 制造轻喜剧
- 让“气人”和“心软”同时存在
- 承载父子旧账和理解难题

写法原则：

- 不写成纯反派
- 不强行洗白
- 要保留他的荒诞、可笑和偶尔温柔

## 女朋友

不是“牺牲者”，而是一个活泼、话多、有审美、有秩序感，和我一起把混乱家庭变成小家的人。

核心特点：

- 活泼
- 话多
- 有审美
- 有生活组织能力
- 能把沉闷场面带起来

叙事功能：

- 小家的秩序来源
- 审美和生活品质来源
- 让家庭重建不只是照护，而是生活

写法原则：

- 写共建能力，不只写感谢和牺牲
- 给她具体动作，不只在结尾夸她
- 不工具人化

## 我

不是“孝顺儿子”，而是一个安静、克制、承担型的人，在照顾父母的过程中完成自救和重建。

核心特点：

- 安静
- 克制
- 承担型
- 不太会现场表达
- 更擅长行动和事后复盘

叙事功能：

- 第一视角和复盘者
- 负责把混乱生活组织成故事
- 承载个人自救线

写法原则：

- 先写事，再写感受
- 不写成完美孝子
- 保留累、气、失败和调整
